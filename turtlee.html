About the Project

This project is an example of using factorization in code to create functions that draw composite shapes to draw a larger scene. I used functions I created to implement the scene that can implement parameters so that they're flexible.

Sample Code

This is an example of how my code utilizes functions to draw the scene. I use parameters to establish reusable functions. 't' is the turtle object, 'x' and 'y' are the horizontal and vertical coordinates. 'balloon_sizes' include the radius and diameter of the balloons and 'balloon_color' is a string indicating the balloon color.

def draw_single_balloon(t, x, y, balloon_sizes, balloon_color):
    # creates size and color changeable function
    offset = 0
    # draw balloon
    for size in balloon_sizes:
        jump_to(t, x + offset, y)
        draw_circle(t, size, fill_color=balloon_color)
    # draw basket
    jump_to(t, (x-70), (y-40))
    draw_rectangle(t, 140, 110, fill_color="brown")
    # draw strings
    jump_to(t, x-70, y-40)
    draw_curve(t, 75, -60, 7)
    jump_to(t, x+15, y+3)
    draw_curve(t, 76, 60, 7)

Final Scene

Here is the completed scene I coded including clouds, hot air balloons, and hills.

This scene depicts two beautiful round green hills and four white cloud clusters. There is also two hot air balloons, one with a red balloon and a yellow balloon. All of this is against a blue sky. It is created by utilizing composite shapes created using turtle graphic functions; circles, rectangles, and curved lines. The clouds are created using layered circles and so are the hills. The balloons have their own function using the previously established functions."
